Урок 1: Введение и первая программа - Ruby

Введение

Ruby — это элегантный и простой в освоении язык программирования, ориентированный на продуктивность разработчиков. Он широко используется для веб-разработки (например, с фреймворком Ruby on Rails), скриптов и автоматизации. В этом уроке вы установите Ruby, настроите среду разработки, напишете свою первую программу и познакомитесь с базовым синтаксисом.

Установка Ruby

Для работы с Ruby необходимо установить интерпретатор. Следуйте этим шагам:

  1. Перейдите на официальный сайт Ruby или используйте менеджер версий, такой как RVM (для Linux/macOS) или RubyInstaller (для Windows).
  2. Скачайте и установите последнюю версию Ruby (например, 3.2 или новее).
  3. Проверьте установку, открыв терминал и выполнив: 
    ruby -v
    Вы должны увидеть версию Ruby, например: ruby 3.2.2.

Рекомендуется использовать текстовый редактор, такой как VS Code, для написания кода.

Первая программа

Создайте файл hello.rb с следующим кодом:

puts "Hello, World!"

Разберём код:

  • puts: Метод, который выводит текст в консоль с добавлением новой строки.
  • "Hello, World!": Строка, которая будет выведена.

Запуск программы

Чтобы запустить программу:

  1. Сохраните файл hello.rb.
  2. Откройте терминал в папке с файлом.
  3. Выполните: 
    ruby hello.rb
    Вывод: Hello, World!

В VS Code вы можете использовать встроенный терминал для запуска (Ctrl+`).

Интерактивная оболочка IRB

Ruby поставляется с интерактивной оболочкой IRB (Interactive Ruby), которая позволяет тестировать код в реальном времени.

  1. Откройте терминал и введите: 
    irb
  2. Попробуйте ввести следующий код: 
    puts "Hello from IRB!"
    Нажмите Enter, и вы увидите вывод: Hello from IRB!.
  3. Для выхода из IRB введите: 
    exit

IRB полезен для экспериментов с небольшими фрагментами кода и изучения возможностей Ruby.

Основы синтаксиса Ruby

Давайте познакомимся с базовыми элементами синтаксиса Ruby, которые пригодятся в дальнейшем:

Комментарии

Комментарии используются для пояснений в коде и игнорируются интерпретатором.

# Это однострочный комментарий
puts "Это выполнится" # Комментарий после кода

=begin
Это многострочный комментарий.
Он может занимать несколько строк.
=end

Переменные

Переменные в Ruby не требуют явного объявления типа. Пример:

name = "Программист"
puts "Привет, #{name}!" # Интерполяция строки

#{...} позволяет вставлять значения переменных в строки.

Базовые операторы

Ruby поддерживает арифметические и другие операторы:

a = 10
b = 5
sum = a + b
puts "Сумма: #{sum}" # Вывод: Сумма: 15

Попробуйте другие операторы: -, *, /, % (остаток от деления).

Программа с вводом данных

Давайте создадим программу, которая запрашивает имя пользователя и выводит персонализированное приветствие.

Создайте файл greet.rb с кодом:

print "Введите ваше имя: "
name = gets.chomp
puts "Привет, #{name}!"

Разберём код:

  • print: Выводит текст без новой строки.
  • gets: Считывает ввод пользователя (включая символ новой строки).
  • chomp: Удаляет символ новой строки из ввода.
  • puts: Выводит приветствие с использованием интерполяции.

Запустите программу: 

ruby greet.rb
Введите имя, например "Алексей", и увидите: Привет, Алексей!.

Практика

Выполните следующие задания, чтобы закрепить материал:

  1. Установите Ruby и проверьте его версию в терминале (ruby -v).
  2. Установите текстовый редактор, например VS Code.
  3. Напишите программу, которая выводит ваше имя в консоль.
  4. Создайте программу, которая выводит "Welcome to Ruby!" и запустите её.
  5. В IRB выполните команду puts "Тестирую IRB!" и проверьте вывод.
  6. Напишите программу, которая запрашивает ваше имя и возраст, затем выводит: "Привет, [имя]! Тебе [возраст] лет."
  7. Создайте программу, которая запрашивает два числа и выводит их сумму.
