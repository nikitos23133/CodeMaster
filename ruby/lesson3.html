Урок 3: Условия, циклы и методы - Ruby

Введение

Условия, циклы и методы — это основы структурного программирования в Ruby. Условия позволяют выполнять код в зависимости от ситуации, циклы — повторять действия, а методы — организовывать код в переиспользуемые блоки. В этом уроке вы изучите эти концепции.

Условные операторы

Ruby поддерживает условные конструкции if, else, elsif и тернарный оператор.

age = 20
if age >= 18
  puts "Вы взрослый"
elsif age >= 13
  puts "Вы подросток"
else
  puts "Вы ребёнок"
end

Тернарный оператор:

status = age >= 18 ? "Взрослый" : "Ребёнок"
puts status

Циклы

Ruby предоставляет несколько способов организации циклов:

Цикл while

count = 1
while count <= 5
  puts "Итерация #{count}"
  count += 1
end

Цикл for

for i in 1..5
  puts "Число #{i}"
end

Метод each для массивов

numbers = [1, 2, 3, 4, 5]
numbers.each do |num|
  puts "Число #{num}"
end

Методы

Методы в Ruby определяются с помощью ключевого слова def. Они могут принимать параметры и возвращать значения.

def greet(name)
  "Привет, #{name}!"
end

puts greet("Алиса") # Привет, Алиса!

Методы с параметрами по умолчанию:

def introduce(name, age = 18)
  puts "Меня зовут #{name}, мне #{age} лет."
end

introduce("Боб")      # Меня зовут Боб, мне 18 лет.
introduce("Алиса", 25) # Меня зовут Алиса, мне 25 лет.

Практика

1. Напишите программу, которая проверяет, является ли число чётным, и выводит результат.

2. Создайте цикл, который выводит числа от 1 до 10.

3. Напишите метод, который принимает два числа и возвращает их сумму.

4. Создайте массив строк и используйте each для их вывода.

5. Напишите программу, которая определяет категорию возраста (ребёнок, подросток, взрослый) с помощью if-elsif.
