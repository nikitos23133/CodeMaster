Урок 4: Функции и модули

Введение

Функции позволяют организовать код в повторно используемые блоки, упрощая его поддержку и чтение. Модули помогают структурировать программу, разделяя код на отдельные файлы. В этом уроке вы научитесь создавать функции, работать с параметрами и организовывать код с помощью модулей.

Создание функций

Функции в Python определяются с помощью ключевого слова def. Они могут принимать параметры и возвращать значения с помощью return.


def greet(name):
    return f"Привет, {name}!"

message = greet("Алиса")
print(message)  # Вывод: Привет, Алиса!

В этом примере функция greet принимает параметр name и возвращает приветственное сообщение.

Параметры и значения по умолчанию

Функции могут иметь параметры с значениями по умолчанию:


def introduce(name, age=18):
    print(f"Меня зовут {name}, мне {age} лет.")

introduce("Боб")  # Вывод: Меня зовут Боб, мне 18 лет.
introduce("Алиса", 25)  # Вывод: Меня зовут Алиса, мне 25 лет.

Если параметр age не указан, используется значение по умолчанию.

Переменное число аргументов

Для функций с переменным числом аргументов используйте *args (для позиционных аргументов) и **kwargs (для именованных):


def sum_numbers(*args):
    return sum(args)

print(sum_numbers(1, 2, 3))  # Вывод: 6
print(sum_numbers(10, 20, 30, 40))  # Вывод: 100

def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Алиса", age=25, city="Страна чудес")
# Вывод:
# name: Алиса
# age: 25
# city: Страна чудес

*args собирает аргументы в кортеж, а **kwargs — в словарь.

Рекурсия

Функция может вызывать саму себя, что называется рекурсией. Пример вычисления факториала:


def factorial(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))  # Вывод: 120 (5 * 4 * 3 * 2 * 1)

Рекурсия полезна для задач, которые можно разделить на подзадачи.

Область видимости переменных

Переменные, определённые внутри функции, имеют локальную область видимости и не доступны снаружи:


def my_function():
    local_var = 10
    print(local_var)

my_function()  # Вывод: 10
# print(local_var)  # Ошибка: local_var не определена

Глобальные переменные доступны внутри функций, но для их изменения используйте global:


count = 0
def increment():
    global count
    count += 1
    print(count)

increment()  # Вывод: 1
increment()  # Вывод: 2

Модули

Модули — это файлы Python (.py), содержащие функции, классы или переменные, которые можно импортировать в другие программы. В редакторе ниже вы можете имитировать модуль, определив функции в коде.

Пример модуля geometry (вместо файла, определите функции в редакторе):


def circle_area(radius):
    import math
    return math.pi * radius ** 2

def rectangle_area(width, height):
    return width * height

Используйте эти функции в коде:


print(circle_area(5))  # Вывод: 78.53981633974483
print(rectangle_area(4, 6))  # Вывод: 24

Стандартные модули Python

Python поставляется с множеством встроенных модулей. Пример использования модуля math:


import math

print(math.sqrt(16))  # Вывод: 4.0
print(math.pi)  # Вывод: 3.141592653589793

Модуль math предоставляет математические функции и константы.
