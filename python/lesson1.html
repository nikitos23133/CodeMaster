Урок 1: Установка Python и первая программа

Что такое Python?

Python — это высокоуровневый язык программирования, известный своей простотой и читаемостью. Созданный в конце 1980-х годов Гуидо ван Россумом, он стал одним из самых популярных языков программирования в мире.

Python применяется в веб-разработке, научных вычислениях, анализе данных, искусственном интеллекте и многих других областях. Его простой синтаксис делает его идеальным выбором для начинающих программистов.

Установка Python

Для начала установите Python на ваш компьютер. Перейдите на официальный сайт Python, чтобы скачать последнюю версию.

Выберите версию, подходящую для вашей операционной системы (Windows, macOS, Linux), и следуйте инструкциям по установке. Для Windows убедитесь, что вы добавили Python в PATH во время установки.

После установки проверьте, что Python работает, открыв терминал (или командную строку) и введя python --version. Если установка прошла успешно, вы увидите текущую версию Python.

Как запустить Python-код

Существует несколько способов запустить Python-код, в зависимости от ваших предпочтений и инструментов.

1. Через командную строку

Сохраните код в файл с расширением .py, например, hello.py, и запустите его командой python hello.py в терминале.

print("Hello, World!")

Вывод:

Hello, World!

2. Используя IDE

Интегрированные среды разработки (IDE) упрощают написание и запуск кода. Популярные IDE для Python:

  • Visual Studio Code — бесплатный редактор с поддержкой Python.
  • PyCharm — мощная IDE для профессиональной разработки.
  • Spyder — IDE для научных вычислений и анализа данных.

Эти IDE позволяют создавать проекты, запускать код и отлаживать его с помощью встроенных инструментов.

3. Онлайн-интерпретаторы

Если установка Python невозможна, используйте онлайн-интерпретаторы для запуска кода в браузере:

  • Replit — онлайн-редактор, поддерживающий Python и другие языки.
  • Jupyter Notebook — интерактивная среда для кода и визуализации данных.

Эти инструменты позволяют тестировать код без локальной установки Python.
