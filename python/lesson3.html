Урок 3: Условные операторы и циклы

Введение

Условные операторы и циклы — это основные конструкции для управления логикой программы. Условные операторы позволяют выполнять код в зависимости от условий, а циклы — повторять действия. В этом уроке вы научитесь использовать if, elif, else, а также циклы for и while.

Условные операторы

Условные операторы позволяют выполнять определённый код, если выполняется заданное условие. В Python используются if, elif и else.


age = 18
if age >= 18:
    print("Вы совершеннолетний!")
else:
    print("Вы несовершеннолетний!")

В этом примере программа проверяет, больше или равно age 18, и выводит соответствующее сообщение.

Использование elif

Для проверки нескольких условий используйте elif (сокращение от "else if"):


score = 85
if score >= 90:
    print("Оценка: A")
elif score >= 80:
    print("Оценка: B")
elif score >= 70:
    print("Оценка: C")
else:
    print("Оценка: D или ниже")

Этот код определяет оценку на основе значения score.

Сложные условия

Вы можете комбинировать условия с помощью логических операторов and, or, not:


temperature = 25
is_sunny = True
if temperature > 20 and is_sunny:
    print("Идём на прогулку!")
else:
    print("Остаёмся дома.")

Здесь прогулка возможна, если температура выше 20 и погода солнечная.

Циклы

Циклы позволяют выполнять код многократно. Python поддерживает два основных типа циклов: for и while.

Цикл for

Цикл for используется для итерации по последовательности, такой как список или диапазон чисел:


for i in range(5):
    print(f"Итерация {i}")

Этот код выведет числа от 0 до 4. Функция range(5) создаёт последовательность от 0 до 4.

Пример с списком:


fruits = ["яблоко", "банан", "вишня"]
for fruit in fruits:
    print(f"Фрукт: {fruit}")

Этот код выведет каждый фрукт из списка.

Цикл while

Цикл while выполняется, пока выполняется условие:


count = 1
while count <= 5:
    print(f"Счёт: {count}")
    count += 1

Этот код выведет числа от 1 до 5. Без count += 1 цикл будет бесконечным!

Операторы break и continue

Оператор break прерывает цикл, а continue пропускает текущую итерацию и переходит к следующей.


for i in range(10):
    if i == 5:
        break  # Прерываем цикл, когда i равно 5
    print(i)

Этот код выведет числа от 0 до 4.


for i in range(5):
    if i % 2 == 0:
        continue  # Пропускаем чётные числа
    print(f"Нечётное: {i}")

Этот код выведет только нечётные числа (1, 3).

Вложенные циклы

Циклы можно вкладывать друг в друга для обработки сложных данных:


for i in range(3):
    for j in range(2):
        print(f"i={i}, j={j}")

Этот код выведет все комбинации i и j, например, i=0, j=0, i=0, j=1, и т.д.
