Урок 5: Работа со списками и словарями

Введение

Списки и словари — это мощные структуры данных в Python, которые позволяют эффективно хранить и обрабатывать информацию. Списки подходят для упорядоченных коллекций, а словари — для данных, организованных в виде пар ключ-значение. В этом уроке вы научитесь создавать, изменять и использовать списки и словари.

Списки

Список — это упорядоченная изменяемая коллекция элементов, которая может содержать данные разных типов. Списки создаются с помощью квадратных скобок [].


numbers = [1, 2, 3, 4]
fruits = ["яблоко", "банан", "вишня"]
mixed = [1, "два", 3.0]
print(numbers)  # Вывод: [1, 2, 3, 4]
print(fruits)   # Вывод: ['яблоко', 'банан', 'вишня']
print(mixed)    # Вывод: [1, 'два', 3.0]

Доступ к элементам и срезы

Элементы списка доступны по индексам (начиная с 0). Отрицательные индексы используются для доступа с конца:


fruits = ["яблоко", "банан", "вишня"]
print(fruits[0])   # Вывод: яблоко
print(fruits[-1])  # Вывод: вишня

Срезы позволяют получить часть списка:


print(fruits[1:3])  # Вывод: ['банан', 'вишня']
print(fruits[:2])   # Вывод: ['яблоко', 'банан']
print(fruits[::2])  # Вывод: ['яблоко', 'вишня'] (каждый второй элемент)

Изменение списков

Списки изменяемы, вы можете заменять элементы или добавлять новые:


fruits = ["яблоко", "банан", "вишня"]
fruits[1] = "апельсин"
print(fruits)  # Вывод: ['яблоко', 'апельсин', 'вишня']

fruits.append("груша")  # Добавление в конец
print(fruits)  # Вывод: ['яблоко', 'апельсин', 'вишня', 'груша']

fruits.remove("вишня")  # Удаление элемента
print(fruits)  # Вывод: ['яблоко', 'апельсин', 'груша']

Методы списков

Вот несколько полезных методов списков:

  • pop(index): Удаляет и возвращает элемент по индексу.
  • sort(): Сортирует список.
  • index(item): Возвращает индекс первого вхождения элемента.

numbers = [3, 1, 4, 1, 5]
numbers.sort()
print(numbers)  # Вывод: [1, 1, 3, 4, 5]

print(numbers.pop(2))  # Вывод: 3
print(numbers)  # Вывод: [1, 1, 4, 5]

print(numbers.index(4))  # Вывод: 2

Вложенные списки

Списки могут содержать другие списки, создавая вложенные структуры:


matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
print(matrix[1])     # Вывод: [4, 5, 6]
print(matrix[1][2])  # Вывод: 6

Словари

Словарь — это неупорядоченная коллекция пар ключ-значение, где ключи уникальны. Словари создаются с помощью фигурных скобок {}.


person = {"name": "Алиса", "age": 25, "city": "Страна чудес"}
print(person)  # Вывод: {'name': 'Алиса', 'age': 25, 'city': 'Страна чудес'}

Доступ к значениям

Значения доступны по ключам. Метод get позволяет избежать ошибок при отсутствии ключа:


person = {"name": "Алиса", "age": 25}
print(person["name"])  # Вывод: Алиса
print(person.get("city", "Не указано"))  # Вывод: Не указано

Изменение словарей

Вы можете добавлять, изменять или удалять пары ключ-значение:


person = {"name": "Алиса", "age": 25}
person["city"] = "Страна чудес"  # Добавление
print(person)  # Вывод: {'name': 'Алиса', 'age': 25, 'city': 'Страна чудес'}

person["age"] = 26  # Изменение
print(person)  # Вывод: {'name': 'Алиса', 'age': 26, 'city': 'Страна чудес'}

del person["city"]  # Удаление
print(person)  # Вывод: {'name': 'Алиса', 'age': 26}

Методы словарей

Полезные методы словарей:

  • keys(): Возвращает все ключи.
  • values(): Возвращает все значения.
  • items(): Возвращает пары ключ-значение.

person = {"name": "Алиса", "age": 25, "city": "Страна чудес"}
print(person.keys())   # Вывод: dict_keys(['name', 'age', 'city'])
print(person.values()) # Вывод: dict_values(['Алиса', 25, 'Страна чудес'])
print(person.items())  # Вывод: dict_items([('name', 'Алиса'), ('age', 25), ('city', 'Страна чудес')])

Итерация по словарю с items():


for key, value in person.items():
    print(f"{key}: {value}")
# Вывод:
# name: Алиса
# age: 25
# city: Страна чудес

Вложенные словари

Словари могут содержать другие словари или списки:


students = {
    "Алиса": {"age": 25, "grades": [90, 85, 88]},
    "Боб": {"age": 30, "grades": [75, 80, 82]}
}
print(students["Алиса"]["grades"])  # Вывод: [90, 85, 88]
print(students["Боб"]["age"])       # Вывод: 30
