Урок 2: Переменные и типы данных в Python

Что такое переменные?

Переменные в Python — это имена, которые используются для хранения данных. Они создаются при присвоении значения с помощью знака =. В Python не нужно заранее объявлять тип переменной, так как он определяется автоматически.

Пример создания переменных:


x = 5
y = "Привет"
z = 3.14
print(x, y, z)

Имена переменных могут содержать буквы, цифры и символ подчеркивания (_), но не могут начинаться с цифры. Например, my_var — допустимое имя, а 2var — нет.

Основные типы данных

Python поддерживает множество типов данных. В этом уроке мы рассмотрим наиболее распространенные:

Целые числа (int)

Целые числа — это числа без дробной части, положительные или отрицательные.


x = 5
y = -10
print(x, y)  # Вывод: 5 -10

Вещественные числа (float)

Вещественные числа содержат дробную часть.


x = 3.14
y = -0.5
print(x, y)  # Вывод: 3.14 -0.5

Строки (str)

Строки — это последовательности символов, заключенные в одинарные или двойные кавычки.


name = "Алиса"
message = 'Привет, мир!'
print(name, message)  # Вывод: Алиса Привет, мир!

Булевы значения (bool)

Булевы значения представляют логические значения: True или False.


is_student = True
has_pet = False
print(is_student, has_pet)  # Вывод: True False

Списки (list)

Списки — это упорядоченные изменяемые коллекции элементов, которые могут быть разных типов.


numbers = [1, 2, 3, 4]
fruits = ["яблоко", "банан", "вишня"]
print(numbers, fruits)  # Вывод: [1, 2, 3, 4] ['яблоко', 'банан', 'вишня']

Словари (dict)

Словари — это неупорядоченные коллекции пар ключ-значение, где ключи уникальны.


person = {"name": "Алиса", "age": 25, "city": "Страна чудес"}
print(person)  # Вывод: {'name': 'Алиса', 'age': 25, 'city': 'Страна чудес'}

Определение типа данных
