Урок 3: Условия, циклы и функции - Swift

Введение

Условия, циклы и функции в Swift позволяют создавать структурированные программы. Условия управляют выполнением кода, циклы повторяют действия, а функции организуют код в переиспользуемые блоки. В этом уроке вы изучите эти концепции.

Условные операторы

Swift использует if, else if, else и тернарный оператор.

let age = 20
if age >= 18 {
    print("Вы взрослый")
} else if age >= 13 {
    print("Вы подросток")
} else {
    print("Вы ребёнок")
}

Тернарный оператор:

let status = age >= 18 ? "Взрослый" : "Ребёнок"
print(status)

Циклы

Swift поддерживает несколько типов циклов.

Цикл for-in

for i in 1...5 {
    print("Итерация \(i)")
}

Цикл while

var count = 1
while count <= 5 {
    print("Итерация \(count)")
    count += 1
}

Цикл по массиву

let numbers = [1, 2, 3, 4, 5]
for num in numbers {
    print("Число \(num)")
}

Функции

Функции в Swift объявляются с помощью func. Они могут возвращать значения.

func greet(name: String) -> String {
    return "Привет, \(name)!"
}

print(greet(name: "Алиса")) // Привет, Алиса!

Функции с параметрами по умолчанию:

func introduce(name: String, age: Int = 18) {
    print("Меня зовут \(name), мне \(age) лет.")
}

introduce(name: "Боб")      // Меня зовут Боб, мне 18 лет.
introduce(name: "Алиса", age: 25) // Меня зовут Алиса, мне 25 лет.

Практика

1. Напишите программу, которая проверяет, является ли число чётным, и выводит результат.

2. Создайте цикл, который выводит числа от 1 до 10.

3. Напишите функцию, которая принимает два числа и возвращает их сумму.

4. Создайте массив строк и используйте цикл for-in для их вывода.

5. Напишите программу, которая определяет категорию возраста (ребёнок, подросток, взрослый) с помощью if-else.
