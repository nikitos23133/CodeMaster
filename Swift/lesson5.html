Урок 5: Структуры и классы - Swift

Введение

Структуры (`struct`) и классы (`class`) в Swift позволяют создавать пользовательские типы данных с свойствами и методами. Структуры подходят для простых данных, а классы поддерживают наследование и используются для сложной логики. В этом уроке вы научитесь определять структуры и классы, создавать экземпляры и добавлять методы для работы с данными.

Работа со структурами

Структуры — это значения, которые копируются при передаче. Создайте файл structs.swift с кодом:

struct Person {
    var name: String
    var age: Int
    
    func greet() -> String {
        return "Привет, я \(name)!"
    }
}

var person = Person(name: "Алексей", age: 25)
print(person.name) // Вывод: Алексей
print(person.greet()) // Вывод: Привет, я Алексей!

// Изменение свойства
person.age = 26
print(person.age) // Вывод: 26

Разберём код:

  • struct Person: Определение структуры с двумя свойствами.
  • func greet: Метод, возвращающий приветствие.
  • var person: Создание экземпляра структуры.
  • Свойства изменяются, если структура объявлена как `var`.

Работа с классами

Классы — это ссылочные типы, которые поддерживают наследование. Попробуйте код в classes.swift:

class Student {
    var name: String
    var grades: [Int]
    
    init(name: String, grades: [Int]) {
        self.name = name
        self.grades = grades
    }
    
    func averageGrade() -> Double {
        let sum = grades.reduce(0, +)
        return Double(sum) / Double(grades.count)
    }
}

let student = Student(name: "Мария", grades: [85, 90, 78])
print(student.name) // Вывод: Мария
print(student.averageGrade()) // Вывод: 84.33333333333333

Разберём код:

  • class Student: Определение класса с свойствами.
  • init: Инициализатор для установки начальных значений.
  • averageGrade: Метод для вычисления средней оценки.
  • let student: Экземпляр класса (свойства можно менять, если они `var`).

Запуск кода

Для выполнения кода используйте Xcode:

  1. Создайте Playground или проект в Xcode.
  2. Скопируйте код из примеров.
  3. Запустите (Cmd+R) и проверьте вывод.
  4. Для файлов используйте терминал: 
    swift structs.swift

Практика

Выполните задания для закрепления:

  1. Создайте структуру `Book` с полями `title` и `year`. Добавьте метод `description`, возвращающий строку с информацией о книге.
  2. Создайте экземпляр `Book` и выведите его описание.
  3. Создайте класс `Car` с полями `model` и `speed`. Добавьте метод `accelerate`, увеличивающий скорость на 10.
  4. Создайте экземпляр `Car`, вызовите `accelerate` и выведите новую скорость.
  5. Создайте структуру `Point` с координатами `x` и `y`. Добавьте метод `distance(to:)` для вычисления расстояния до другой точки.
  6. Создайте класс `Team` с массивом имён участников и методом для вывода их количества.
  7. Напишите программу, использующую структуру и класс для моделирования студента и его оценок.

Подробности о структурах и классах читайте в документации Swift.
