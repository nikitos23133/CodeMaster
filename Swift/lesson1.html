Урок 1: Введение и первая программа - Swift

Введение

Swift — это мощный и интуитивно понятный язык программирования от Apple, используемый для разработки приложений для iOS, macOS, watchOS и tvOS. Он сочетает производительность и безопасность. В этом уроке вы установите Xcode, настроите среду разработки и напишете свою первую программу на Swift.

Установка Xcode

Swift лучше всего разрабатывать в Xcode, который доступен только на macOS:

  1. Откройте App Store на macOS и найдите Xcode.
  2. Скачайте и установите Xcode (версия 15 или новее).
  3. Запустите Xcode и согласитесь с условиями лицензии.
  4. Проверьте установку Swift, создав новый проект или выполнив в терминале: 
    swift --version
    Вывод: Swift version 5.x.

Для написания кода можно также использовать VS Code с расширением Swift, но Xcode предпочтительнее для начинающих.

Первая программа

Создайте файл hello.swift или используйте Xcode для создания новой программы:

print("Hello, World!")

Разберём код:

  • print: Функция, которая выводит текст в консоль.
  • "Hello, World!": Строка для вывода.

Запуск программы

Варианты запуска:

  1. Через терминал:
    • Сохраните файл hello.swift.
    • Откройте терминал в папке с файлом.
    • Выполните: 
      swift hello.swift
      Вывод: Hello, World!
  2. Через Xcode:
    • Откройте Xcode и создайте новый проект ("Command Line Tool").
    • В файле main.swift напишите код.
    • Нажмите "Run" (Cmd+R) для запуска.

Практика

1. Установите Xcode и проверьте версию Swift в терминале.

2. Создайте новый проект в Xcode для Command Line Tool.

3. Напишите программу, которая выводит ваше имя в консоль.

4. Создайте программу, которая выводит "Welcome to Swift!" и запустите её через терминал или Xcode.
