Урок 3: Условия, циклы и методы - C#

Введение

Условия, циклы и методы в C# позволяют создавать структурированные программы. Условия управляют выполнением кода, циклы повторяют действия, а методы организуют код в переиспользуемые блоки. В этом уроке вы изучите эти концепции.

Условные операторы

C# использует if, else if, else и тернарный оператор.


using System;

class Program {
    static void Main(string[] args) {
        int age = 20;
        if (age >= 18) {
            Console.WriteLine("Вы взрослый");
        } else if (age >= 13) {
            Console.WriteLine("Вы подросток");
        } else {
            Console.WriteLine("Вы ребёнок");
        }
    }
}

Тернарный оператор:


string status = age >= 18 ? "Взрослый" : "Ребёнок";
Console.WriteLine(status);

Циклы

C# поддерживает несколько типов циклов.

Цикл for


using System;

class Program {
    static void Main(string[] args) {
        for (int i = 1; i <= 5; i++) {
            Console.WriteLine($"Итерация {i}");
        }
    }
}

Цикл while


using System;

class Program {
    static void Main(string[] args) {
        int count = 1;
        while (count <= 5) {
            Console.WriteLine($"Итерация {count}");
            count++;
        }
    }
}

Цикл foreach


using System;
using System.Collections.Generic;

class Program {
    static void Main(string[] args) {
        List numbers = new List {1, 2, 3, 4, 5};
        foreach (int num in numbers) {
            Console.WriteLine($"Число {num}");
        }
    }
}

Методы

Методы в C# объявляются внутри классов и могут возвращать значения.


using System;

class Program {
    static string Greet(string name) {
        return $"Привет, {name}!";
    }

    static void Main(string[] args) {
        Console.WriteLine(Greet("Алиса")); // Привет, Алиса!
    }
}

Методы с параметрами по умолчанию:


using System;

class Program {
    static void Introduce(string name, int age = 18) {
        Console.WriteLine($"Меня зовут {name}, мне {age} лет.");
    }

    static void Main(string[] args) {
        Introduce("Боб");      // Меня зовут Боб, мне 18 лет.
        Introduce("Алиса", 25); // Меня зовут Алиса, мне 25 лет.
    }
}

Практика

Попробуйте выполнить следующие задания в редакторе кода ниже:

  1. Напишите программу, которая проверяет, является ли число чётным, и выводит результат.
  2. Создайте цикл, который выводит числа от 1 до 10.
  3. Напишите метод, который принимает два числа и возвращает их сумму.
  4. Создайте список строк и используйте foreach для их вывода.
  5. Напишите программу, которая определяет категорию возраста (ребёнок, подросток, взрослый) с помощью if-else.

Введите код в редактор, нажмите "Запустить код" и проверьте результат. Если возникнет ошибка, наш помощник предложит исправление. Если сервер недоступен, скопируйте код и запустите локально с помощью dotnet run.
