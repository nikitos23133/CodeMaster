Урок 2: Переменные, типы данных и операторы - C#

Введение

C# — это строго типизированный язык, где переменные требуют явного указания типа. В этом уроке вы изучите переменные, основные типы данных и операторы в C# для выполнения вычислений и манипуляций с данными.

Переменные

Переменные в C# объявляются с указанием типа данных.


using System;

class Program {
    static void Main(string[] args) {
        string name = "Алиса";
        int age = 25;
        Console.WriteLine($"Имя: {name}, Возраст: {age}");
    }
}

Переменные должны быть инициализированы перед использованием.

Типы данных

C# поддерживает примитивные и ссылочные типы данных:

  • int, long: Целые числа (например, 42).
  • float, double: Числа с плавающей точкой (например, 3.14).
  • string: Строки (например, "Hello").
  • bool: Логические значения (true, false).
  • List: Списки (например, new List {1, 2, 3}).

using System;
using System.Collections.Generic;

class Program {
    static void Main(string[] args) {
        int number = 42;
        double pi = 3.14;
        string greeting = "Привет!";
        bool isActive = true;
        List numbers = new List {1, 2, 3};
        Console.WriteLine($"{number}, {pi}, {greeting}, {isActive}, {string.Join(",", numbers)}");
    }
}

Операторы

Операторы в C# позволяют выполнять вычисления и сравнения.

Арифметические операторы


using System;

class Program {
    static void Main(string[] args) {
        int a = 10;
        int b = 3;
        Console.WriteLine(a + b); // 13
        Console.WriteLine(a - b); // 7
        Console.WriteLine(a * b); // 30
        Console.WriteLine(a / b); // 3
        Console.WriteLine(a % b); // 1
    }
}

Операторы сравнения


using System;

class Program {
    static void Main(string[] args) {
        int x = 5;
        int y = 3;
        Console.WriteLine(x == y); // false
        Console.WriteLine(x != y); // true
        Console.WriteLine(x > y);  // true
        Console.WriteLine(x <= y); // false
    }
}

Логические операторы


using System;

class Program {
    static void Main(string[] args) {
        bool isAdult = true;
        bool hasTicket = false;
        Console.WriteLine(isAdult && hasTicket); // false
        Console.WriteLine(isAdult || hasTicket); // true
        Console.WriteLine(!isAdult);            // false
    }
}
