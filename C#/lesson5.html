Урок 5: Классы и объекты - C#

Введение

Классы и объекты — основа объектно-ориентированного программирования (ООП) в C#. Классы определяют шаблон для объектов, которые содержат данные (поля) и поведение (методы). В этом уроке вы научитесь создавать классы, экземпляры и методы для моделирования реальных сущностей.

Создание класса

Класс определяет свойства и методы. Создайте файл Classes.cs с кодом:


using System;

class Person {
    public string Name { get; set; }
    public int Age { get; set; }

    public Person(string name, int age) {
        Name = name;
        Age = age;
    }

    public string Greet() {
        return $"Привет, я {Name}!";
    }
}

class Program {
    static void Main() {
        Person person = new Person("Алексей", 25);
        Console.WriteLine(person.Name); // Вывод: Алексей
        Console.WriteLine(person.Greet()); // Вывод: Привет, я Алексей!

        person.Age = 26;
        Console.WriteLine(person.Age); // Вывод: 26
    }
}

Разберём код:

  • class Person: Определение класса.
  • public string Name { get; set; }: Свойство с геттером и сеттером.
  • Person: Конструктор для инициализации.
  • Greet: Метод, возвращающий приветствие.

Использование объектов

Объекты — экземпляры классов. Попробуйте код в Students.cs:


using System;
using System.Collections.Generic;

class Student {
    public string Name { get; set; }
    public List Grades { get; set; }

    public Student(string name, List grades) {
        Name = name;
        Grades = grades;
    }

    public double AverageGrade() {
        int sum = 0;
        foreach (int grade in Grades) {
            sum += grade;
        }
        return (double)sum / Grades.Count;
    }
}

class Program {
    static void Main() {
        Student student = new Student("Мария", new List { 85, 90, 78 });
        Console.WriteLine(student.Name); // Вывод: Мария
        Console.WriteLine(student.AverageGrade()); // Вывод: 84.33333333333333
    }
}

Разберём код:

  • Student: Класс с полем для списка оценок.
  • AverageGrade: Метод для вычисления среднего.
  • new Student: Создание объекта.

Запуск кода

Для выполнения кода используйте .NET SDK или Visual Studio:

  1. Создайте файл (Classes.cs или Students.cs).
  2. В терминале выполните: 
    
dotnet run Classes.cs
  3. Или откройте в Visual Studio и запустите (F5).

Практика

Выполните задания:

  1. Создайте класс Book с полями Title и Year. Добавьте метод Description.
  2. Создайте объект Book и выведите его описание.
  3. Создайте класс Car с полями Model и Speed. Добавьте метод Accelerate.
  4. Создайте объект Car, увеличьте скорость и выведите результат.
  5. Создайте класс Rectangle с полями Width и Height. Добавьте метод Area.
  6. Создайте класс Team с полем Members (список имён) и методом для вывода их количества.
  7. Напишите программу с классом Employee, хранящим имя и зарплату, и методом для повышения зарплаты.
