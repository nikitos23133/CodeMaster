Урок 1: Введение и первая программа - C#

Введение

C# — это универсальный язык программирования от Microsoft, используемый для разработки приложений, веб-сервисов, игр (с Unity) и многого другого. Он сочетает мощь и простоту. В этом уроке вы установите .NET SDK, настроите среду разработки и напишете свою первую программу.

Установка .NET SDK

Для работы с C# необходимо установить .NET SDK:

  1. Перейдите на официальный сайт .NET.
  2. Скачайте и установите .NET SDK (например, .NET 8 или новее).
  3. Проверьте установку, открыв терминал и выполнив: 
    
dotnet --version
    Вывод: 8.0.x или другая версия.

Рекомендуется установить Visual Studio Community (Windows/macOS) или VS Code с расширением C# для разработки.

Первая программа

Ниже пример программы, которая выводит "Hello, World!":


using System;

class Program {
    static void Main(string[] args) {
        Console.WriteLine("Hello, World!");
    }
}

Разберём код:

  • using System;: Импортирует базовые функции .NET.
  • class Program: Определяет класс программы.
  • static void Main: Точка входа программы.
  • Console.WriteLine: Выводит текст в консоль.

Компиляция и запуск

В редакторе ниже вы можете писать и запускать C# код. Для локальной разработки:

  1. Сохраните код в файл HelloWorld.cs.
  2. Откройте терминал в папке с файлом.
  3. Выполните: 
    
dotnet new console -o HelloWorld
    Переместите HelloWorld.cs в папку HelloWorld.
  4. Перейдите в папку: 
    
cd HelloWorld
  5. Запустите: 
    
dotnet run
    Вывод: Hello, World!

В Visual Studio создайте новый проект "Console App" и запустите с помощью "Run".
