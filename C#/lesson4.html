Урок 4: Работа с коллекциями - C#

Введение

Коллекции в C# — это мощные структуры данных для хранения и обработки наборов элементов. Наиболее распространённые коллекции — это List<T> (динамический список) и Dictionary<TKey, TValue> (ассоциативный массив). В этом уроке вы научитесь создавать, изменять и использовать списки и словари в C#.

Работа с List<T>

List<T> — это динамический массив, который может расти или сжиматься. Создайте файл Lists.cs с кодом:


using System;
using System.Collections.Generic;

class Program {
    static void Main() {
        List<string> fruits = new List<string> { "Яблоко", "Банан", "Апельсин" };
        Console.WriteLine(string.Join(", ", fruits)); // Вывод: Яблоко, Банан, Апельсин

        // Добавление элемента
        fruits.Add("Груша");
        Console.WriteLine(string.Join(", ", fruits)); // Вывод: Яблоко, Банан, Апельсин, Груша

        // Доступ и изменение
        fruits[1] = "Манго";
        Console.WriteLine(fruits[1]); // Вывод: Манго

        // Итерация
        foreach (string fruit in fruits) {
            Console.WriteLine($"Фрукт: {fruit}");
        }
    }
}

Разберём код:

  • List<string>: Список строк.
  • Add: Добавляет элемент в конец.
  • fruits[1]: Доступ по индексу.
  • foreach: Перебор элементов.

Работа с Dictionary<TKey, TValue>

Dictionary<TKey, TValue> хранит пары ключ-значение. Попробуйте код в Dictionaries.cs:


using System;
using System.Collections.Generic;

class Program {
    static void Main() {
        Dictionary<string, int> scores = new Dictionary<string, int> {
            { "Алексей", 85 },
            { "Мария", 92 },
            { "Иван", 78 }
        };
        Console.WriteLine($"Оценка Марии: {scores["Мария"]}"); // Вывод: 92

        // Добавление/обновление
        scores["Ольга"] = 95;
        Console.WriteLine(string.Join(", ", scores)); // Вывод: [Алексей, 85], [Мария, 92], [Иван, 78], [Ольга, 95]

        // Итерация
        foreach (var pair in scores) {
            Console.WriteLine($"{pair.Key}: {pair.Value}");
        }
    }
}

Разберём код:

  • Dictionary<string, int>: Словарь с ключами-строками и значениями-целыми числами.
  • scores["Ольга"]: Добавление/обновление пары.
  • foreach ... KeyValuePair: Перебор пар.

Запуск кода

Для выполнения кода используйте .NET SDK:

  1. Создайте файл (Lists.cs или Dictionaries.cs).
  2. Откройте терминал в папке с файлом.
  3. Скомпилируйте и запустите: 
    
dotnet run Lists.cs
  4. Или используйте Visual Studio для запуска проекта.

Практика

Попробуйте выполнить следующие задания в редакторе кода ниже:

  1. Создайте List<string> с названиями 5 городов и выведите их.
  2. Добавьте новый город и замените третий город.
  3. Создайте Dictionary<string, int> с 3 именами и возрастами.
  4. Обновите возраст одного человека и добавьте нового.
  5. Напишите программу, выводящую имена и возраста в формате "Имя: Возраст".
  6. Создайте List<int> с числами и выведите только числа больше 10.
  7. Создайте словарь с продуктами и их ценами, затем выведите продукт с самой высокой ценой.

Введите код в редактор, нажмите "Запустить код" и проверьте результат. Если возникнет ошибка, наш помощник предложит исправление. Если сервер недоступен, скопируйте код и запустите локально с помощью dotnet run.

Предложенное исправление:

Попробуйте изменить код для других заданий (например, фильтрация чисел или поиск самого дорогого продукта) или введите свои варианты.
