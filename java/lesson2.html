Урок 2: Переменные, типы данных и операторы

Введение

Переменные, типы данных и операторы — основа программирования на Java. Переменные хранят данные, типы данных определяют, какие значения могут быть использованы, а операторы позволяют выполнять операции. В этом уроке вы научитесь объявлять переменные, работать с типами данных и использовать операторы.

Переменные

В Java переменные объявляются с указанием типа данных. Они должны быть инициализированы перед использованием.

public class Variables {
    public static void main(String[] args) {
        int age = 25;
        String name = "Алиса";
        System.out.println("Имя: " + name + ", Возраст: " + age);
    }
}

В этом примере объявлены переменные age (тип int) и name (тип String).

Типы данных

Java — строго типизированный язык, где каждая переменная имеет определённый тип. Основные типы данных делятся на примитивные и ссылочные.

Примитивные типы

  • byte: 8-битное целое число (-128 до 127).
  • short: 16-битное целое число (-32,768 до 32,767).
  • int: 32-битное целое число (-2^31 до 2^31-1).
  • long: 64-битное целое число (добавьте L, например, 100L).
  • float: 32-битное число с плавающей точкой (добавьте f, например, 3.14f).
  • double: 64-битное число с плавающей точкой (например, 3.14).
  • char: 16-битный символ Unicode (например, 'A').
  • boolean: Логическое значение (true, false).
public class DataTypes {
    public static void main(String[] args) {
        int number = 42;
        double pi = 3.14;
        char letter = 'A';
        boolean isActive = true;
        System.out.println(number + ", " + pi + ", " + letter + ", " + isActive);
    }
}

Ссылочные типы

Ссылочные типы включают классы, массивы и интерфейсы. Наиболее распространённый пример — String:

String greeting = "Привет, Java!";
System.out.println(greeting);

Операторы

Операторы позволяют выполнять операции с данными. Основные категории:

Арифметические операторы

Используются для математических вычислений:

public class Arithmetic {
    public static void main(String[] args) {
        int a = 10;
        int b = 3;
        System.out.println(a + b); // 13
        System.out.println(a - b); // 7
        System.out.println(a * b); // 30
        System.out.println(a / b); // 3
        System.out.println(a % b); // 1
    }
}

Операторы сравнения

Сравнивают значения и возвращают true или false:

public class Comparison {
    public static void main(String[] args) {
        int x = 5;
        int y = 3;
        System.out.println(x == y); // false
        System.out.println(x != y); // true
        System.out.println(x > y);  // true
        System.out.println(x <= y); // false
    }
}

Логические операторы

Комбинируют условия:

public class Logical {
    public static void main(String[] args) {
        boolean isAdult = true;
        boolean hasTicket = false;
        System.out.println(isAdult && hasTicket); // false
        System.out.println(isAdult || hasTicket); // true
        System.out.println(!isAdult);            // false
    }
}

Операторы присваивания

Присваивают значения:

public class Assignment {
    public static void main(String[] args) {
        int x = 10;
        x += 5; // x = x + 5
        System.out.println(x); // 15
        x *= 2; // x = x * 2
        System.out.println(x); // 30
    }
}

Практика

1. Создайте программу, которая объявляет переменные для имени, возраста и роста, и выводит их.

2. Напишите программу, которая выполняет арифметические операции с двумя числами (сложение, умножение, деление).

3. Создайте программу, которая проверяет, является ли число чётным, используя оператор %.

4. Напишите программу, которая использует логические операторы для проверки, находится ли число в диапазоне от 1 до 100.

5. Объявите переменные разных примитивных типов и выведите их значения.
