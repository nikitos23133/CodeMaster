Урок 1: Установка JDK и первая программа

Что такое Java?

Java — это объектно-ориентированный язык программирования, разработанный в 1995 году компанией Sun Microsystems (ныне часть Oracle). Он широко используется для создания надежных и масштабируемых приложений, включая веб-приложения, мобильные приложения для Android, корпоративные системы и многое другое. Java известна своей платформонезависимостью благодаря Java Virtual Machine (JVM), которая позволяет запускать программы на любой платформе.

Установка JDK

Для начала вам нужно установить Java Development Kit (JDK) на ваш компьютер. Перейдите на официальный сайт Oracle или используйте OpenJDK с сайта OpenJDK.

Выберите версию JDK, подходящую для вашей операционной системы (Windows, macOS, Linux), и следуйте инструкциям по установке. Убедитесь, что вы добавили JDK в переменную PATH вашей системы.

После установки проверьте, что Java работает. Откройте терминал или командную строку и введите:

        javac -version
        java -version

Если JDK установлен правильно, вы увидите версии javac и java.

Как запустить Java-код

Существует несколько способов запустить Java-код, в зависимости от ваших инструментов.

1. Через командную строку

Как описано ниже, создайте файл с расширением .java, скомпилируйте его с помощью javac и запустите с помощью java.

2. Используя IDE

Интегрированные среды разработки (IDE) упрощают написание и запуск кода. Популярные IDE для Java:

  • IntelliJ IDEA — мощная IDE для профессиональной разработки.
  • Eclipse — бесплатная и популярная IDE для Java.
  • NetBeans — еще одна бесплатная IDE с поддержкой Java.

Эти IDE позволяют создавать проекты, компилировать и запускать код с помощью встроенных инструментов.

3. Онлайн-интерпретаторы

Для тестирования кода без установки JDK используйте онлайн-интерпретаторы:

  • Replit — онлайн-редактор для Java и других языков.
  • JDoodle — онлайн-компилятор для Java.

Первая программа на Java

Теперь давайте напишем нашу первую программу на Java. Создайте новый файл с помощью текстового редактора, например, Visual Studio Code, и назовите его Hello.java.

В файле напишите следующий код:

        public class Hello {
            public static void main(String[] args) {
                System.out.println("Hello, World!");
            }
        }

Сохраните файл, откройте терминал, перейдите в директорию с файлом и выполните:

        javac Hello.java
        java Hello

Вы увидите вывод:

        Hello, World!

Поздравляем, вы написали свою первую программу на Java!

Структура программы на Java

Каждая программа на Java должна быть частью класса. В примере выше мы определили класс с именем Hello. Каждый класс в Java должен содержать метод main, который является точкой входа в программу. Метод main всегда имеет следующую сигнатуру:

        public static void main(String[] args)

Это означает, что метод main является публичным (public), статическим (static), не возвращает значения (void) и принимает массив строк (String[] args) в качестве аргументов.

Переменные и типы данных

В Java переменные должны быть объявлены с указанием их типа данных, так как Java — это статически типизированный язык. Основные типы данных включают:

  • int: Целые числа, например, 42.
  • double: Числа с плавающей точкой, например, 3.14.
  • String: Строки, например, "Привет".
  • boolean: Булевы значения, true или false.

Пример:

        public class Variables {
            public static void main(String[] args) {
                int age = 25;
                String name = "Боб";
                System.out.println("Имя: " + name + ", Возраст: " + age);
            }
        }

В этом примере мы объявили переменные age типа int и name типа String и вывели их значения.

Пример с числами

Создадим программу, которая выполняет арифметические операции с числами:

        public class Calculator {
            public static void main(String[] args) {
                double x = 10.5;
                double y = 5.2;
                System.out.println("Сумма: " + (x + y));
                System.out.println("Разность: " + (x - y));
                System.out.println("Произведение: " + (x * y));
            }
        }

Скомпилируйте и запустите программу:

        javac Calculator.java
        java Calculator

Вывод будет:

        Сумма: 15.7
        Разность: 5.3
        Произведение: 54.6
