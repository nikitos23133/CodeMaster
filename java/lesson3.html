Урок 3: Классы и объекты

Введение

Объектно-ориентированное программирование (ООП) — это основа Java. Классы и объекты позволяют моделировать реальные сущности, такие как люди, машины или книги. В этом уроке вы научитесь создавать классы, определять их свойства и методы, а также использовать объекты для взаимодействия с классами.

Что такое классы и объекты?

Класс — это шаблон или чертёж для создания объектов. Объект — это экземпляр класса, содержащий конкретные данные и поведение, определённые в классе.

public class Person {
    // Поля (свойства)
    String name;
    int age;

    // Метод
    void introduce() {
        System.out.println("Меня зовут " + name + ", мне " + age + " лет.");
    }
}

В этом примере класс Person описывает человека с именем и возрастом.

Создание объектов

Чтобы использовать класс, создайте объект с помощью ключевого слова new:

public class Main {
    public static void main(String[] args) {
        Person person = new Person();
        person.name = "Алиса";
        person.age = 25;
        person.introduce(); // Вывод: Меня зовут Алиса, мне 25 лет.
    }
}

Объект person — это экземпляр класса Person.

Конструкторы

Конструктор — это специальный метод, который вызывается при создании объекта. Он используется для инициализации полей:

public class Person {
    String name;
    int age;

    // Конструктор
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    void introduce() {
        System.out.println("Меня зовут " + name + ", мне " + age + " лет.");
    }
}

public class Main {
    public static void main(String[] args) {
        Person person = new Person("Боб", 30);
        person.introduce(); // Вывод: Меня зовут Боб, мне 30 лет.
    }
}

Ключевое слово this используется для обращения к полям объекта.

Методы

Методы определяют поведение объектов. Они могут принимать параметры и возвращать значения:

public class Person {
    String name;
    int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    String getGreeting() {
        return "Привет, я " + name + "!";
    }
}

public class Main {
    public static void main(String[] args) {
        Person person = new Person("Алиса", 25);
        System.out.println(person.getGreeting()); // Вывод: Привет, я Алиса!
    }
}

Практика

1. Создайте класс Car с полями для марки и года выпуска, и методом для вывода информации об автомобиле.

2. Добавьте в класс Car конструктор для инициализации полей.

3. Создайте класс Book с полями для названия и автора, и методом, возвращающим описание книги.

4. Напишите программу, создающую два объекта класса Person и вызывающую их методы.
