Урок 4: Работа с DOM

Введение

Document Object Model (DOM) — это программный интерфейс, который позволяет JavaScript взаимодействовать с HTML-документом. С помощью DOM вы можете выбирать элементы, изменять их содержимое, стили и реагировать на действия пользователя. В этом уроке вы научитесь работать с DOM для создания интерактивных веб-страниц.

Что такое DOM?

DOM представляет HTML-документ как дерево объектов, где каждый тег — это узел. JavaScript может манипулировать этими узлами, изменяя структуру, содержимое или стили страницы.



Привет, мир!

JavaScript может получить доступ к этому элементу и изменить его:

const div = document.getElementById("myDiv");
console.log(div.textContent);  // Вывод: Привет, мир!

Выборка элементов

Для работы с DOM сначала нужно выбрать элементы. Основные методы:

  • getElementById(id): Выбирает элемент по ID.
  • querySelector(selector): Выбирает первый элемент по CSS-селектору.
  • querySelectorAll(selector): Выбирает все элементы по CSS-селектору.
const title = document.getElementById("title");
const paragraphs = document.querySelectorAll("p");
const firstPara = document.querySelector(".intro");

console.log(title);       // Вывод: 
...

console.log(paragraphs);  // Вывод: NodeList с 
 элементами
console.log(firstPara);   // Вывод: 
...

Изменение содержимого и стилей

Вы можете изменять текст, HTML-содержимое или стили элементов:

const div = document.getElementById("myDiv");
div.textContent = "Новый текст";  // Изменение текста
div.innerHTML = "Жирный текст";  // Изменение HTML

div.style.backgroundColor = "lightblue";  // Изменение стиля
div.style.fontSize = "20px";

textContent изменяет только текст, а innerHTML позволяет добавлять HTML-разметку.

Работа с атрибутами

Методы getAttribute и setAttribute позволяют работать с атрибутами элементов:

const img = document.querySelector("img");
img.setAttribute("src", "new-image.jpg");
console.log(img.getAttribute("src"));  // Вывод: new-image.jpg

Обработка событий

События позволяют реагировать на действия пользователя, такие как клики или ввод текста. Используйте addEventListener:

const button = document.getElementById("myButton");
button.addEventListener("click", () => {
    alert("Кнопка нажата!");
});

const input = document.querySelector("input");
input.addEventListener("input", (event) => {
    console.log(`Введено: ${event.target.value}`);
});

В примере кнопка вызывает всплывающее окно при клике, а поле ввода выводит текст в консоль при изменении.

Практика

1. Создайте HTML-страницу с заголовком и измените его текст с помощью JavaScript.

2. Найдите все элементы с классом item и измените их цвет фона.

3. Создайте кнопку, которая при нажатии меняет текст параграфа.

4. Добавьте изображение на страницу и измените его атрибут src с помощью JavaScript.

5. Создайте поле ввода, которое при вводе текста обновляет содержимое другого элемента в реальном времени.
