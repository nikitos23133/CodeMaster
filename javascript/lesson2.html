Урок 2: Переменные, типы данных и операторы

Введение

Переменные, типы данных и операторы — это основа программирования в JavaScript. Переменные используются для хранения данных, типы данных определяют, какие значения могут быть сохранены, а операторы позволяют выполнять операции с этими данными. В этом уроке вы научитесь объявлять переменные, работать с различными типами данных и использовать операторы.

Переменные

В JavaScript переменные объявляются с помощью ключевых слов var, let или const. Переменные хранят данные, которые можно использовать и изменять в программе.

let name = "Алиса";
const age = 25;
var city = "Страна чудес";

console.log(name);  // Вывод: Алиса
console.log(age);   // Вывод: 25
console.log(city);  // Вывод: Страна чудес

let используется для переменных, которые могут изменяться. const — для неизменяемых значений. var — устаревший способ, но всё ещё встречается в старом коде.

Изменение переменной:

let score = 10;
score = 20;
console.log(score);  // Вывод: 20

const pi = 3.14;
// pi = 3.14159;  // Ошибка: нельзя изменить const

Типы данных

JavaScript — это динамически типизированный язык, где тип данных определяется автоматически. Основные типы данных:

  • number: Числа (целые или дробные).
  • string: Строки (текст).
  • boolean: Логические значения (true, false).
  • null: Отсутствие значения.
  • undefined: Переменная объявлена, но не имеет значения.
  • object: Объекты (включая массивы).
let num = 42;               // number
let text = "Привет";        // string
let isStudent = true;       // boolean
let nothing = null;         // null
let notAssigned;            // undefined
let person = { name: "Боб", age: 30 };  // object
let numbers = [1, 2, 3];    // array (тип object)

console.log(typeof num);    // Вывод: number
console.log(typeof text);   // Вывод: string
console.log(typeof person); // Вывод: object

Оператор typeof возвращает тип данных переменной.

Числа (number)

Числа могут быть целыми или дробными. JavaScript поддерживает арифметические операции:

let x = 10;
let y = 3.14;
console.log(x + y);  // Вывод: 13.14

Строки (string)

Строки заключаются в одинарные или двойные кавычки. Их можно конкатенировать с помощью +:

let greeting = "Привет, ";
let name = "Алиса";
console.log(greeting + name);  // Вывод: Привет, Алиса

Шаблонные строки (с использованием `) позволяют вставлять выражения:

let age = 25;
console.log(`Мне ${age} лет`);  // Вывод: Мне 25 лет

Объекты и массивы

Объекты — это коллекции пар ключ-значение, а массивы — упорядоченные списки:

let person = { name: "Алиса", age: 25 };
console.log(person.name);  // Вывод: Алиса

let fruits = ["яблоко", "банан", "вишня"];
console.log(fruits[1]);  // Вывод: банан

Операторы

Операторы позволяют выполнять операции с данными. Основные категории:

Арифметические операторы

Используются для математических вычислений:

let a = 10;
let b = 3;
console.log(a + b);  // Вывод: 13 (сложение)
console.log(a - b);  // Вывод: 7 (вычитание)
console.log(a * b);  // Вывод: 30 (умножение)
console.log(a / b);  // Вывод: 3.333... (деление)
console.log(a % b);  // Вывод: 1 (остаток)
console.log(a ** b); // Вывод: 1000 (возведение в степень)

Операторы сравнения

Сравнивают значения и возвращают true или false:

let x = 5;
let y = "5";
console.log(x == y);   // Вывод: true (нестрогое сравнение)
console.log(x === y);  // Вывод: false (строгое сравнение)
console.log(x != y);   // Вывод: false (нестрогое неравенство)
console.log(x !== y);  // Вывод: true (строгое неравенство)
console.log(x > 3);    // Вывод: true
console.log(x <= 5);   // Вывод: true

=== проверяет значение и тип, в отличие от ==.

Логические операторы

Используются для комбинирования условий:

let isAdult = true;
let hasTicket = false;
console.log(isAdult && hasTicket);  // Вывод: false (И)
console.log(isAdult || hasTicket);  // Вывод: true (ИЛИ)
console.log(!isAdult);              // Вывод: false (НЕ)

Операторы присваивания

Присваивают значения переменным:

let x = 10;
x += 5;  // То же, что x = x + 5
console.log(x);  // Вывод: 15
x *= 2;  // То же, что x = x * 2
console.log(x);  // Вывод: 30

Практика

1. Создайте переменные с вашим именем, возрастом и городом, используя let и const. Выведите их в консоль.

2. Создайте массив из трёх ваших любимых блюд и выведите второй элемент.

3. Создайте объект с информацией о книге (название, автор, год) и выведите название книги.

4. Напишите программу, которая выполняет арифметические операции с двумя числами (сложение, вычитание, умножение).

5. Используйте typeof, чтобы проверить типы переменных: числа, строки, объекта и неопределённой переменной.
