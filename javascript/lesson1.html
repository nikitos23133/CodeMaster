Урок 1: Введение в JavaScript

Что такое JavaScript?

JavaScript — это язык программирования, который позволяет добавлять интерактивность к веб-страницам. Он является одной из трех основных технологий веб-разработки, наряду с HTML и CSS. HTML структурирует содержимое, CSS отвечает за его стилизацию, а JavaScript делает страницы динамичными и интерактивными.

Созданный в 1995 году Бренданом Айком, JavaScript изначально назывался LiveScript, но был переименован для маркетинговых целей из-за популярности Java. Несмотря на название, JavaScript и Java — это совершенно разные языки. JavaScript широко используется для создания веб-приложений, игр, анимаций и даже серверных приложений с помощью Node.js.

Как запустить JavaScript-код

Существует несколько способов запустить JavaScript-код, в зависимости от ваших целей и инструментов.

1. Через браузер

Как показано выше, вы можете включить JavaScript в HTML-файл и открыть его в браузере. Консоль браузера покажет вывод вашего кода.

2. Используя Node.js

JavaScript также можно запускать вне браузера с помощью Node.js. Установите Node.js, создайте файл с расширением .js (например, hello.js) и добавьте код:

        console.log("Hello, World!");

Откройте терминал, перейдите в директорию с файлом и выполните:

        node hello.js

Вы увидите вывод "Hello, World!" в терминале.

3. Онлайн-интерпретаторы

Если установка не требуется, используйте онлайн-интерпретаторы:

  • Replit — онлайн-редактор для JavaScript и других языков.
  • JSFiddle — инструмент для тестирования JavaScript, HTML и CSS.

Эти платформы позволяют писать и запускать код прямо в браузере.

Переменные и типы данных

В JavaScript переменные используются для хранения данных. Вы объявляете переменную с помощью ключевых слов let или const. JavaScript — это динамически типизированный язык, поэтому тип данных переменной определяется автоматически.

Пример:

        let name = "Алиса";
        console.log(name); // Выводит: Алиса

Основные типы данных в JavaScript включают:

  • Строки: Последовательности символов, например, "Привет".
  • Числа: Целые числа или числа с плавающей точкой, например, 42 или 3.14.
  • Булевы значения: true или false.
  • Массивы: Упорядоченные коллекции значений, например, [1, 2, 3].
  • Объекты: Коллекции ключей и значений, например, { name: "Алиса", age: 25 }.

Пример с числами

Создадим программу, которая выполняет простые арифметические операции:

        let a = 5;
        let b = 10;
        console.log("Сумма:", a + b); // Выводит: Сумма: 15
        console.log("Разность:", a - b); // Выводит: Разность: -5
        console.log("Произведение:", a * b); // Выводит: Произведение: 50

Здесь мы используем операторы +, - и * для выполнения арифметических операций.
