Урок 2: Переменные, типы данных и операторы - Go

Введение

Go — это строго типизированный язык, где переменные требуют явного указания типа или автоматического вывода. В этом уроке вы изучите переменные, основные типы данных и операторы в Go для выполнения вычислений и манипуляций с данными.

Переменные

Переменные в Go объявляются с помощью var или короткого синтаксиса :=.


package main

import "fmt"

func main() {
    var name string = "Алиса"
    age := 25 // Короткий синтаксис
    fmt.Printf("Имя: %s, Возраст: %d\n", name, age)
}

:= используется только внутри функций и автоматически определяет тип.

Типы данных

Go поддерживает следующие основные типы данных:

  • int, uint: Целые числа (например, 42).
  • float32, float64: Числа с плавающей точкой (например, 3.14).
  • string: Строки (например, "Hello").
  • bool: Логические значения (true, false).
  • []int, []string: Слайсы (динамические массивы).

package main

import "fmt"

func main() {
    number := 42
    pi := 3.14
    greeting := "Привет!"
    isActive := true
    numbers := []int{1, 2, 3}
    fmt.Println(number, pi, greeting, isActive, numbers)
}

Операторы

Операторы в Go позволяют выполнять вычисления и сравнения.

Арифметические операторы


package main

import "fmt"

func main() {
    a := 10
    b := 3
    fmt.Println(a + b) // 13
    fmt.Println(a - b) // 7
    fmt.Println(a * b) // 30
    fmt.Println(a / b) // 3
    fmt.Println(a % b) // 1
}

Операторы сравнения


package main

import "fmt"

func main() {
    x := 5
    y := 3
    fmt.Println(x == y) // false
    fmt.Println(x != y) // true
    fmt.Println(x > y)  // true
    fmt.Println(x <= y) // false
}

Логические операторы


package main

import "fmt"

func main() {
    isAdult := true
    hasTicket := false
    fmt.Println(isAdult && hasTicket) // false
    fmt.Println(isAdult || hasTicket) // true
    fmt.Println(!isAdult)            // false
}
