Урок 1: Введение и первая программа - Go

Введение

Go (или Golang) — это современный язык программирования, разработанный для простоты, надёжности и высокой производительности. Он популярен для серверных приложений, облачных систем и микросервисов. В этом уроке вы установите Go, настроите среду разработки и напишете свою первую программу.

Установка Go

Для работы с Go необходимо установить компилятор. Следуйте этим шагам:

  1. Перейдите на официальный сайт Go.
  2. Скачайте установщик для вашей операционной системы (например, Go 1.21 или новее).
  3. Установите Go, следуя инструкциям. На Windows и macOS установка автоматическая, на Linux распакуйте архив в /usr/local.
  4. Настройте переменную GOPATH (по умолчанию — ~/go):
    • Windows: Добавьте %USERPROFILE%\go\bin в Path.
    • Mac/Linux: Добавьте в ~/.bashrc или ~/.zshrc: 
      
export GOPATH=$HOME/go
export PATH=$PATH:$GOPATH/bin:/usr/local/go/bin
  5. Проверьте установку, выполнив: 
    
go version
    Вывод: go version go1.21.x.

Рекомендуется использовать VS Code с расширением Go для удобной разработки.

Первая программа

Создайте файл hello.go с следующим кодом:


package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

Разберём код:

  • package main: Определяет основной пакет, точка входа программы.
  • import "fmt": Подключает пакет для форматирования и вывода.
  • func main(): Главная функция, с которой начинается выполнение.
  • fmt.Println: Выводит текст в консоль.

Компиляция и запуск

Чтобы скомпилировать и запустить программу:

  1. Сохраните файл hello.go.
  2. Откройте терминал в папке с файлом.
  3. Выполните: 
    
go run hello.go
    Вывод: Hello, World!
  4. Для компиляции в исполняемый файл: 
    
go build hello.go
    Затем запустите: ./hello (Linux/macOS) или hello.exe (Windows).
